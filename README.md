A simple web application that allows users to search for Pokémon by name or ID number, using the PokeAPI.
The app displays detailed information about each Pokémon, including its image, name, stats, moves, and abilities. 
Was created with with no external js or css libaries.

This project was created as part of a school course assignment to explore API usage and frontend development.

Features:
    Search by name or ID
    Displayed image, stats, moves and abilities
    Tries to look like a pokedex
